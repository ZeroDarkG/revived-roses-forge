# revived-roses-forge
A massive nostalgia expansion for Forge 1.21.11. Beyond the classic Cyan Rose and Small Paeonia, this mod restores Ruby ore with a full tier of armor and tools. Includes functional Gears, the decorative Scarecrow, and the new Blue Rose Bush. Fully updated to the latest 1.21.11 registry systems (setId) for a seamless modern experience.
